# Serverless Contact Form Project

## Struktur
- index.html
- api/contact.js
- vercel.json

## Cara Deploy
1. Upload folder ke GitHub
2. Import ke Vercel
3. Isi Environment Variables:
   - SUPABASE_URL
   - SUPABASE_KEY
   - RESEND_API_KEY
   - EMAIL_FROM=onboarding@resend.dev
   - EMAIL_TO=emailmu@gmail.com
